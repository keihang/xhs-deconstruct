# 博主概览模板

为每个博主创建概览文件，格式如下：

```markdown
# {博主名} — 博主概览

## 基本信息

| 项目 | 信息 |
|------|------|
| 昵称 | {博主名} |
| 小红书号 | {小红书号} |
| IP属地 | {IP属地} |
| 简介 | {个人简介} |
| 粉丝数 | {粉丝数} |
| 获赞与收藏 | {总获赞} |

## 内容定位

- **赛道**：{内容领域}
- **核心工具/话题**：{主要涉及的工具或话题}
- **目标人群**：{目标受众}
- **账号内容形式**：{图文/视频/混合}
- **本次分析样本**：最近半年图文帖子

## Top 10 笔记数据

| # | 标题 | 点赞 | 收藏 | 评论 |
|---|------|------|------|------|
| 1 | {标题} | {点赞} | {收藏} | {评论} |
| ... | ... | ... | ... | ... |
```

**数据来源**：从 `opencli xiaohongshu user` 获取博主信息和帖子列表；先按最近半年与 `type` 过滤图文，再按点赞排序取前20篇候选，最终汇总成功抓取的最多10篇图文样本数据。
