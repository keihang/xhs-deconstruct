# 质量自检清单

🛑 **STOP：完成所有步骤后，逐项检查以下清单，全部通过才算完成。**

## 完整性检查
- [ ] 博主概览是否完整（粉丝量、内容类型、更新频率）
- [ ] 候选池是否严格基于最近半年内的图文帖子建立
- [ ] 每篇图文帖子是否都有原文+拆解，且首图信息已被纳入封面分析
- [ ] 总结是否覆盖所有维度（标题、封面、内容、数据、模式）
- [ ] 可复用模板是否足够具体（能直接套用）

## 数据检查
- [ ] 是否先按 `type` 过滤图文，再按点赞排序取前20候选
- [ ] 互动数据是否准确（点赞、收藏、评论）
- [ ] 收藏率、互动率是否计算正确
- [ ] 标题公式是否有数据验证
- [ ] 内容结构是否有案例支撑

## 格式检查
- [ ] 文件命名是否符合规范（无空格、序号两位数）
- [ ] Obsidian图片语法是否正确
- [ ] 图片引用路径是否包含帖子ID子目录
- [ ] 表格格式是否正确
- [ ] 链接是否有效
- [ ] HTML 报告是否可在浏览器正常打开

## 实用性检查
- [ ] 标题公式是否能直接套用
- [ ] 内容结构是否能直接套用
- [ ] 封面模板是否能直接套用
- [ ] 选题方向是否足够具体
